# Python Dashboard

Quick and practical dashboard for data analysis of common occurrences at the testing plant.

## Features

### 1. Binary Viewer
Tool for effortlessly loading and analysing historical data.

- Export plots as PDF  
- Multiple subplots  
- Autoscaling  
- Hover values  
- CSV export  

### 2. Chemical Analysis *(WIP)*
Early-stage tool for rapid visualisation of chemical analysis data.

- Error plotting for Lambda  
- Ostwald triangle visualisation  
- Additional analysis tools in development  

### 3. Quicktrends *(Planned)*
Future feature focused on real-time monitoring.

- OPC server integration  
- Live quicktrend visualisation  
- Alarm system based on derivatives and thresholds  
- Unified interface
