# Python Dashboard

Quick and practical dashboard for data analysis of common occurrences at the testing plant.

## Features

### 1. Binary Viewer
Tool for effortlessly loading and analysing historical data.

- Export plots as PDF  
- Multiple subplots  
- Autoscaling  
- Hover values  
- CSV export  

### 2. Consumption Calculator
Allows for calculation and export of consumption data for budgeting

### 3. Quicktrends *(WIP)*
Future feature focused on real-time monitoring.

- OPC server integration  
- Live quicktrend visualisation  
- Alarm system based on derivatives and thresholds  
- Unified interface
